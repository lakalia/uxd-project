SDETool (Simple Data Exploration Tool)

The SDETool is a Simple Data Exploration Tool designed to be hooked up to any data set for simple, quick and easy data exploration.

  • Just input the necessary data in the form to the right, hit the submit button and start exploring!
  • If security is your concern, download the tool to your local machine where only you have access to the data you connect to -- no internet connection required.
The SDETool is a work in progress. Initially it was set up to visualize Hans Rosling's Gapminder population growth data which is the default data set onload. As such, currently, once the form below is actually working, whatever data set is hooked up to the tool must have a Year column and data points by country. Also currently, the 'Download' tab will simply expose the directory of files for the tool which must all be saved down individually. Eventually, javascript will be added to create a zip file of the directory for download.
